Validator Help

The HED Validator is an online-tool designed to make validation of HED tags easy. To start, put your HED tags in either an Excel file or a tab-separated text file. (Example files can be found here.) Once you have a file, you are ready to begin the validation.

Step 1: Specify the file to be validated

Indicate the name of the file to validate, using the Browse button to navigate in your file system:

Step 2: Define the locations of the HED tags in your file.

After you have selected a file, HED Validator will try to automatically fill in the remaining information into its form based on the contents of your file. In the following example, the file EmotionEventInformation.xlsx was selected. If a spreadsheet has multiple worksheets, HED Validator selects the first one by default (ImaginedEmotion), but provides a drop down for you to select a different one.

By default, HED Validator assumes that the first row gives the names of the columns and uses these names to guess which columns correspond to special tags such as Event/Category or Event/Label. In the above example the HED Validator has guessed that the the second column (entitled Category) holds the Event/Category tag. The Validator was also able to correctly guess the Event/Label and Event/Description columns.
